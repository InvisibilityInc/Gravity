Gravity Version 1.7.0

Release Notes

New Feature: Audio Recording

Feature: You can now record your screen and mic audio and chat with it directly, enabling seemless support for meetings, lectures, and more!

New Feature: Menu Bar Icon

Feature: We've added a new menu bar icon that allows you to quickly access the app and start recording.

New Feature: Advanced Settings

Feature: We've added a new tab in the settings menu that allows you to customize your LLM experience.

Performance Improvements: Improved Energy Usage

Performance: We've optimized the app to load the exact resources it needs, only when it needs it, reducing memory usage and improving battery life.

Performance Improvements: Longer Contexts

Performance: We've implemented a dynamic sliding window to increase the maximum length of handled context (text input), allowing for more complex and longer interactions.

Bug Fixes: Stability Improvements

Bug Fix: Resolved several critical crashes that were impacting the user experience. The app is now more stable, ensuring a smoother and reliable performance.
