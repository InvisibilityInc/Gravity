Gravity Version 1.8.0

Release Notes

New Features: Enhanced Stop Token Handling

Feature: Improved user experience with the implementation of multiple stop token handling. Now supports seamless integration of different stopping criteria for better command control.

New Features: Sampling Enhancements

Feature: Introduced new sampling options: repeat_last_n, repeat_penalty, frequency_penalty, and presence_penalty. These provide finer control over the behavior of the language model's outputs.

New Features: Model Quality and Interaction

Feature: Upgraded to OpenHermes 2.5, which includes enhancements to the light mode view of the audio widget, ensuring a more enjoyable interaction.

New Features: Joyful Download Status

Feature: Revamped the download UI to cleanly display the status, adding a touch of joy to the user's experience during model downloads.

Performance Improvements: Sliding Context Window

Performance: Integrated a dynamic sliding window mechanism to handle longer context lengths, boosting the potential for more complex interactions.

Bug Fixes: Stop Token Integration

Bug Fix: Addressed and fixed the integration issue with the stop function, ensuring commands are executed as expected.

Bug Fixes: Buffer Size Correction

Bug Fix: Corrected a potential buffer size issue to guarantee token count stays within the maximum limit, enhancing overall stability.

Upgrades: Merge with Upstream

Update: Merged with upstream changes, focusing on improvements to the model downloader.

Thank You!

We sincerely appreciate your continued use of Gravity. Our team is dedicated to ongoing improvements and delivering the best user experience. Should you have any feedback or encounter issues, please reach out to us at support@grav.ai. Your input is invaluable to our development process.
